## ðŸ’¡ Doctor Tips

### Before Submitting Your Account:

1. **Double-check your account number**
   - Enter it twice
   - Compare with your bank statement

2. **Use your full registered name**
   - Must match your doctor registration

   - Swift code for international banks

   - Visit Bank Accounts page
   - Look for verification status
   - Usually verified within 24 hours

   - Have a backup account
   - One will be set as default
   - Switch default anytime

3. **View full account number**
   - Click "Show" to reveal full number
   - Verify it's correct
   - Edit if you spot an error

## ðŸ” Troubleshooting

### Doctor: My Account is Taking Too Long to Verify

**Possible reasons**:
- Admin team reviewing details carefully
- Account name doesn't match registration
- Account number appears invalid
- Additional verification needed

**What to do**:
1. Check your account details are correct
2. Ensure account name matches your registration
3. Contact admin if >24 hours with no update
4. Check for any emails from admin team

### Doctor: I Made a Mistake in My Account Number

**Solution**:
1. Add the correct account as a new account
2. Wait for verification of new account
3. Delete the incorrect account
4. Set correct account as default

**OR** (if already has payments):
1. Contact admin immediately
2. Request update to account details
3. Admin can verify correct account
4. Avoid submitting duplicate accounts

### Admin: Account Name Doesn't Match

**What to do**:
1. Check doctor's registration details
2. Verify legal name vs preferred name
3. Contact doctor for clarification
4. Update if necessary
5. Only verify when confirmed match

### Admin: Suspicious Account Details

**Red flags**:
- Account name completely different from doctor name
- Recently changed account multiple times
- Bank account in different person's name
- Unusual bank or location

**Action**:
1. Do NOT verify immediately
2. Contact doctor for verification
3. Request additional documentation
4. Verify identity thoroughly
5. Only approve when satisfied
